# optionsBotByBit

Бот предназначен для **сканирования опционов на бирже Bybit** и **построения перспективных опционных спредов** на основе текущих рыночных условий.

## Возможности

- Сканирование рынка опционов Bybit.
- Поиск/подбор потенциально интересных опционных спредов.
- Интеграция с Telegram (бот) для получения сигналов/уведомлений.

## Требования

- Node.js `>= 16`
- pnpm

## Установка

```bash
pnpm install
```

## Запуск (dev)

```bash
pnpm run dev
```

## Сборка

```bash
pnpm run build
```

## Запуск собранной версии

```bash
pnpm run preview
```

## Переменные окружения

Проект использует `dotenv`. Создай файл `.env` и укажи необходимые значения (например, токен Telegram-бота и ключи Bybit, если они используются в твоей конфигурации).
